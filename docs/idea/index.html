# 介绍

书籍是人类进步的阶梯，IntelliJ IDEA是猿类快速上树的阶梯。
读书的时候用的是MyEclipse，工作后用了一段时间eclipse，后来接触了idea，才算真正的解放劳动力，谁用谁说好的工具。
