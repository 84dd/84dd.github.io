## .less 编译成小程序支持的 .wxss
通过`File Watchers`中添加`less`来实现
![less](/WebStorm/less.png)
其中有三个参数需要注意的
```
// 该路径为安装less所在的位置可执行文件lessc的路径，windown下为lessc.exe
Program = /usr/local/bin/lessc

Arguments = FileName$

Output path to refresh = $FileNameWithoutExtension$.wxss
```
