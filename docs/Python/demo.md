## Hello world
长期以来，编程界都认为刚接触一门新语言时，如果首先使用它来编写一个在屏幕上显示消息“Hello world!”的程序，将给你带来好运。要使用Python来编写这种Hello World程序，只需一行代码:
``` python
print("Hello world!")
```
> 以上诙谐的语言来自书籍《Python编程：从入门到实践》
