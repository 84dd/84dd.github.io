# Windows 环境

由于博主使用的是Mac，所以Window下的环境是在虚拟机上完成的。如有差异，请在留言中反馈。

# 查看是否已安装

python -V
# 'python' 不是内部或外部命令，也不是可运行的程序或批处理文件。
1
2

# 下载

Windows 下的安装包可以去官网下载最新版本，请注意自己使用的是32位还是64位机器。 我下载的是Python 3.7.4 64位安装版 国内下载太慢了，我把这个exe安装包放在git上，可以在这里下载

# 安装

点击安装包一路默认安装就可以了，不过记得勾上设置环境变量，如果忘记勾选了，可以重新安装一次。

# 再次查看

python -V
# Python 3.7.4
1
2

如上显示 Python 3.7.4 就是安装成功了
