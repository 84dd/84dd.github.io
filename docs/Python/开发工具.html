# Sublime Text 3

Sublime Text是一款轻量但功能非常强大的文本编辑器，吊打Windows的Note。我们去官网下载吧http://www.sublimetext.com/ (opens new window) 下载完然后一路默认安装，你可以体验到心情非常舒适。

# 配置支持python3

Sublime Text内置了很多编译器

列表中显示了很多语言，其中包括Python，很可惜经过试验，这里的Python是2.x版本，下面我们让它支持3.x吧

首先，我们要知道Python3的安装目录，但是之前说过的安装是默认安装，并且自动添加环境变量的设置，从来没有选择过目录，如何得知安装到哪里呢？

# Mac

# MacOs
type -a python

# python is an alias for python3
# python is /usr/local/bin/python
# python is /usr/bin/python
1
2
3
4
5
6

通过命令，我们知道MacOs中的python在目录/usr/local/bin/python下 启动Sublime Text，并选择菜单Tools ▶ Build System ▶ New Build System，将下面代码覆盖到新建的文件里，并保存到默认目录，命名为Python3.sublime-build

{
"cmd": ["/usr/local/bin/python3", "-u", "$file"],
}
1
2
3

再次选择菜单Tools ▶ Build System ▶ Python3，将刚才添加的编译器设置默认编译器

通过命令并分析环境变量，我们知道Windows中的python在目录C:\Users\lym\AppData\Local\Programs\Python\Python37\下 启动Sublime Text，并选择菜单Tools ▶ Build System ▶ New Build System，将下面代码覆盖到新建的文件里，并保存到默认目录，命名为Python3.sublime-build

{
	"cmd": ["C:/Users/lym/AppData/Local/Programs/Python/Python37/python.exe", "-u", "$file"]
}

再次选择菜单Tools ▶ Build System ▶ Python3，将刚才添加的编译器设置默认编译器

# 使用

例子：启动Sublime Text，新建文件，并保存到任意目录，在文件中添加以下代码

print("Hello world!")
1

选择菜单Tools ▶ Build (快捷键Mac: Command+ B, Windows: Ctrl+ B)
