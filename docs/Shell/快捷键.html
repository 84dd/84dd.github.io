# 快捷键

仅在Mac下测试

  1. Ctrl a 将光标移动到行首 (a 是 ahead 的缩写)

  2. Ctrl e 将光标移动到行尾 (e 是 end 的缩写)

  3. Ctrl r 进入历史命令查找窗口，输入要查找的命令可快速选择历史命令 (r 是 retrieve 检索的缩写)， 也可以通过history | grep xxx来检索历史。使用iTerm2的朋友可以试下自动提示命令

  4. Ctrl w 删除当前光标附近的一个词 (以空格隔开的字符串 w 是 word的缩写)

  5. Ctrl u 删除命令行上已经敲出来的所有文本 (即删除整行 u 是 undo的缩写)

  6. Ctrl y 粘贴 Ctrl w、Ctrl u 等命令删除的文本

  7. Ctrl l 清屏 等于 clear

  8. !xx关键字 列出最近一条包含有 xx关键字 的历史命令。例如最近打了一个很长很长， 很难很难的命令，例如echo 'I love you!' > /dev/null 可以通过向上剪头查找，也可以类似!ec这样查找。 如果命令比较久远，可以通过history查找后!行号进行调用。
